For my lab, I decided to use cellular automata for my automatic composition. I started by using p5.js for the visualization. Specifically, I used this code, an implementation of Conway's Game of Life. I chose to use Conway's game of life for its emergent properties of gliders and ring like structures based on simple rules. Especially for the gliders, I like to think they are creatures moving, so I thought it would be interesting to sonify. Originally, I programmed a function to map the x,y position of live cells in the grid to a frequency. I played around with different linear and sinusoidal functions, but none of these produced something appealing to the ear. Eventually I landed on hard coding the scales to 4 chords across 4 octaves to give the Game of Life the quality of improvising over a scale as you would if you were playing music.

The code works by playing a note for the length of the framerate every time a cell is born.
